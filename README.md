# 16-bit Arithmetic Logic Unit (ALU)

## Overview

This project implements a 16-bit Arithmetic Logic Unit (ALU) using Verilog HDL with a hierarchical RTL design approach. The ALU performs arithmetic, logical, increment, decrement, and comparison operations by integrating reusable hardware modules. The design is verified through simulation using a dedicated testbench.

## Features

- 16-bit arithmetic operations
- Addition using a 4-bit Ripple Carry Adder hierarchy
- Subtraction
- Increment and Decrement
- Bitwise AND, OR, XOR, and NOT operations
- 16-bit comparison
- Modular and reusable RTL design
- Simulation verified using a Verilog testbench

## Tools Used

- Verilog HDL
- Xilinx Vivado
- ModelSim (or your simulator)
